# Advanced React Portfolio Project

## Overview

This project is the final capstone for the Coursera Advanced React course. It demonstrates the ability to design and build a responsive, production-ready React application using modern best practices and component-driven architecture.

The application incorporates form validation, reusable UI components, state management patterns, and accessibility considerations while maintaining clean, readable code.

---

## Key Concepts Demonstrated

- Functional components and React Hooks  
- Component composition and reusable design patterns  
- Form management using Formik  
- Schema-based validation using Yup  
- UI component styling with Chakra UI  
- Controlled components and state management  
- Conditional rendering and dynamic feedback  
- Responsive layout design  
- Basic accessibility best practices  

---

## Lab Notes

- Built using a modular component structure for scalability.
- Implemented Formik to manage form state, submission handling, and validation lifecycle.
- Used Yup schemas to enforce structured input validation and improve reliability.
- Leveraged Chakra UI for consistent styling, theme management, and responsive design.
- Focused on clean separation of concerns between UI, logic, and validation.
- Emphasized readable, maintainable code for long-term extensibility.
